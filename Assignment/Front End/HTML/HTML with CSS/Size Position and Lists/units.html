CSS Units Examples

This div uses em: Font size is 2em (2 times the parent's font size of 16px).
This div uses rem: Font size is 2rem (2 times the root font size of 20px).
This div uses vh: Height is 50vh (50% of the viewport height).
This div uses vw: Width is 50vw (50% of the viewport width).
This div uses %: Width is 50% of the parent element's width.
